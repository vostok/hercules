# Vostok Hercules
Event based transport subsystem of Vostok

## How to Build
You need to have Java and [Maven](https://maven.apache.org) installed.

Hercules requires Maven 3.3.9 or higher. Java 8 is used to build project.

### Build jars
```
mvn package
```
